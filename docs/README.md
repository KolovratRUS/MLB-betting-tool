# MLB Total Runs Screening Tool

A specialized screening tool for identifying the strongest Over opportunities in MLB games, specifically targeting Over 5.5, 6.5, 7.5, and 8.5 run totals.

## Overview

The MLB Total Runs Screening Tool ranks today's MLB games by their likelihood of producing high total runs. It leverages advanced statistical analysis of current season data, recent trends, home/away splits, and starting pitcher impact to help identify the best Over opportunities for building 2-leg and 3-leg multi-leg betting parlays.

## Features

- **Game Ranking**: Rank today's MLB games by likelihood of exceeding run total thresholds
- **Multi-Leg Building**: Identify optimal 2-leg and 3-leg combinations for parlay bets
- **Advanced Scoring**: Consider season stats, 30-game trends, home/away splits, and pitcher data
- **Real-Time Data**: Pull current team statistics and starting pitcher information
- **Clean Dashboard**: Easy-to-read interface showing ranked games with key metrics

## Supported Run Totals

- Over 5.5 runs
- Over 6.5 runs
- Over 7.5 runs
- Over 8.5 runs
